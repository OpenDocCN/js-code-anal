## 下载

### Docker

```
docker pull apachecn0/js-code-anal
docker run -tid -p <port>:80 apachecn0/js-code-anal
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g js-code-anal
js-code-anal <port>
# 访问 http://localhost:{port} 查看文档
```
